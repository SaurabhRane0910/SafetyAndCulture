# Safety & Culture — Single Page Website

This is a creative, animated, fully responsive **single-page** site with:
- **Home** hero (headline + short description + CTA)
- **Pricing** (3 pricing cards + features)
- **Contact** (Name/Email/Message form with basic validation)
- **Smooth scrolling navigation** and scroll-reveal animations

## Run locally

Option A (simplest): open `index.html` in your browser.

Option B (recommended): run a local static server.

If you have Node.js:
```bash
npx serve .
```

## Deploy (get a live link)

### Netlify (fastest)
- Drag & drop this folder into the Netlify dashboard “Sites” page.
- You’ll get a live URL immediately.

### GitHub Pages
- Create a new GitHub repo and upload these files (`index.html`, `styles.css`, `script.js`).
- In **Settings → Pages**, set source to your main branch.
- Your live link will appear on that page.

### Vercel
- Create a new project, import the folder/repo as a **static** site.
- Deploy and copy the generated URL.
